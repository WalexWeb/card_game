
# Card Game API

> Современный сервер на Node.js для онлайн-карточной игры с использованием WebSocket (Socket.IO) и Express.

## Возможности
- Поддержка многопользовательских партий
- Роли игроков: "слова" и "значения"
- Реализация раундов, подсчёт очков, автоматическое завершение игры
- Веб-сокет события для управления игрой в реальном времени

## Технологии
- Node.js
- Express
- Socket.IO
- ES-модули
- Helmet, CORS, Compression

## Быстрый старт

1. **Клонируйте репозиторий и перейдите в папку проекта:**
	```sh
	git clone <repo-url>
	cd card_game
	```

2. **Установите зависимости:**
	```sh
	npm install
	```

3. **Создайте файл `.env` и укажите порт:**
	```env
	PORT=3000
	```

4. **Запустите сервер:**
	```sh
	npm start
	```

## Структура проекта

```
src/
  app.js                # Точка входа, настройка сервера
  models/
	 game.js             # Логика игры
  socket/
	 gameHandler.js      # Обработка событий Socket.IO
  utils/
	 deckGenerator.js    # Генерация колоды
```

## Основные WebSocket события

| Событие           | Описание                        |
|-------------------|---------------------------------|
| join_game         | Присоединение к игре            |
| select_word       | Выбор слова                     |
| select_meaning    | Выбор значения                  |
| next_round        | Переход к следующему раунду     |
| reset_game        | Начать новую игру               |
| leave_game        | Покинуть игру                   |

## REST API

- `GET /` — Проверка работы сервера
- `GET /health` — Проверка статуса
